# [가나디백과] 쇼츠 콘텐츠 퀄리티 컨트롤(QC) 체크리스트

콘텐츠를 제작하고 업로드하기 전, 유튜브 알고리즘의 선택을 받고 '재사용된 콘텐츠' 정책 위반을 피하기 위해 반드시 확인해야 할 체크리스트입니다.

## 1. 팩트 소싱 (Source Verification)
- [ ] 단순 블로그/커뮤니티 글이 아닌 **공신력 있는 전문가/기관**의 자료를 참고했는가?
  - *추천 소스: AKC(미국애견협회), VCA, 설채현 수의사, 이찬종 소장, 동물행동의학 칼럼 등*
- [ ] 강아지의 행동에 대한 원인이 '과학적/심리적'으로 타당하게 설명되었는가?
- [ ] 참고한 원본 글을 그대로 복사하지 않고, 채널 톤앤매너에 맞게 '재해석(Curation)' 하였는가?

## 2. 영상 소스 컷편집 (Manual Curation)
- [ ] 기계적인 컷편집이 아닌, **직접(수동으로)** 대본과 가장 잘 어울리는 10~15초 하이라이트 구간을 골라냈는가?
- [ ] 강아지의 특정 행동(갸우뚱, 하품, 짖음 등) 타이밍이 대본의 '펀치라인(반전/핵심 내용)'과 정확히 일치하는가?
- [ ] 시청자의 시선을 끌 수 있는 '웃기거나 귀여운' 킬링 파트가 영상 초반 3초 이내에 존재하는가?

## 3. 재사용된 콘텐츠 정책 방어 (Policy Adherence)
- [ ] 남의 영상을 의미 없이 이어 붙인 **'단순 짜깁기'**가 아님을 확신하는가?
- [ ] 원본 영상에는 없는 나만의 **'새로운 교육적 가치(행동 심리 해설)'** 또는 **'독창적인 스토리텔링(상황극)'**이 추가되었는가?
- [ ] 영상 위에 텍스트 자막, 화면 줌인/줌아웃, 효과음 등 시청자의 이해를 돕기 위한 **'시각적 재창조'** 편집이 들어갔는가?
- [ ] (중요) AI 음성(TTS)이 남의 글을 영혼 없이 읽어주기만 하는 영상이 아닌가? (가나디백과는 TTS 배제 및 자막 중심 권장)

## 4. UI/UX 및 포맷 규격 (Format Compliance)
- [ ] **상단 고정 훅(Hook) 자막**: 시선을 끄는 텍스트가 18자 이내로 영상 내내 상단에 고정되어 있는가?
- [ ] **하단 롤링 자막**: 강아지 속마음이나 행동 심리 팩트가 화면 하단에 가독성 좋게 전환되고 있는가?
- [ ] **글로벌 타겟팅**: 메인 한글 자막 아래에 작은 폰트로 영문 자막이 병기되어 있는가?
- [ ] **행동 유도(CTA)**: 영상 마지막 반전/클라이맥스 구간에 '구독/좋아요'를 유도하는 텍스트나 장치가 들어있는가?

## 5. 자주 쓰는 더블체크된 행동학 팩트 (Verified Facts Reference)
*이 섹션은 시안이 수의학 및 행동학 전문가(AKC, VCA 등)의 교차 검증을 거친 팩트들만 지속적으로 업데이트하는 공간입니다. 기획 시 언제든 꺼내어 쓰세요!*

- [x] **꼬리 흔들기**: 무조건 기분 좋다는 뜻이 아님. 오른쪽으로 치우치면 긍정(반가움), 왼쪽으로 치우치면 부정(경계/불안). 꼬리가 빳빳하게 서서 흔들리면 극도의 긴장/공격성 주의. *(검증: AKC)*
- [x] **하품하기**: 졸릴 때도 하지만, 낯선 환경에서 스스로 '스트레스/긴장'을 풀고 상대방에게 "난 공격 의사가 없어(카밍 시그널)"라고 말하는 행동. *(검증: VCA Hospitals)*
- [x] **고개 갸우뚱하기**: 사람의 말을 더 정확히 듣기 위해 귓바퀴 방향을 조절하거나, 주둥이 너머로 사람의 입모양과 표정을 더 잘 읽기 위한 고도의 집중 행동. *(검증: 수의학 칼럼)*
- [x] **앞발 한쪽 들고 정지**: 사냥개 본능에서 나온 '포인팅(Pointing)' 자세이거나, 다음 상황을 예측하며 고도로 집중/긴장할 때 나오는 행동. *(검증: 설채현 수의사)*
- [x] **미친 듯이 뛰어다니기 (우다다/줌미)**: 'FRAPs'라는 정상 현상. 스트레스나 에너지를 발산하는 긍정적 행동. *(검증: AKC)*
- [x] **보호자에게 몸 기대기**: 서열 우위가 아님. 체온을 나누며 신뢰를 표현하거나, 불안할 때 위안을 얻기 위한 스킨십. *(검증: AKC)*
- [x] **자신의 발을 집착적으로 핥기**: 애정 표현과 다름. 알레르기, 통증, 혹은 극도의 스트레스를 해소하기 위한 강박 행동일 확률이 높음. *(검증: VCA Hospitals)*
- [x] **땅이 꺼져라 한숨 쉬기**: 눈을 반쯤 감고 쉬면 '편안함/만족감'. 두 눈을 똑바로 뜨고 쳐다보며 쉬면 '실망감(놀이 끝) 혹은 불만'의 의미. *(검증: AKC)*
- [x] **화장실까지 졸졸 따라오기**: 무리(Pack) 동물 본능. 무방비 상태인 보호자를 지켜주려는 행동이거나 가벼운 분리불안. *(검증: 수의학 칼럼)*
- [x] **배변 후 흙을 세게 차는 뒷발차기**: 변을 숨기려는 게 아님. 발바닥 패드의 페로몬을 사방에 퍼뜨려 자신의 구역을 강하게 표시(마킹)하는 위풍당당한 행동. *(검증: VCA Hospitals)*
- [x] **배를 훌러덩 까고 자기**: 야생에서는 볼 수 없는 취약한 자세. 환경과 보호자를 10,000% 신뢰하여 완벽히 안심했거나 체온을 낮추려는 행동. *(검증: AKC)*
